Installing
==========

Stable library from PyPi:

-  Just run ``sudo pip install veml6075``

Latest/development library from GitHub:

-  ``git clone https://github.com/pimoroni/veml6075-python``
-  ``cd veml6075-python``
-  ``sudo ./install.sh``
